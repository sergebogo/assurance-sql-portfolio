# Projet SQL – Pilotage des dossiers assurance-vie

## Contexte
Ce projet simule le pilotage de dossiers d’assurance-vie dans un contexte
bancaire français (back-office, relation client, service juridique).

Il s’inspire des problématiques opérationnelles rencontrées dans les
établissements bancaires et assurantiels en France.

## Objectifs métier
- Suivre le backlog de dossiers
- Analyser les délais de traitement
- Identifier les dossiers hors SLA
- Aider à la prise de décision managériale

## Modèle de données
- dossiers : cycle de vie des dossiers assurance
- sla : délais contractuels par service
- beneficiaires : informations sur les ayants droit

## Indicateurs clés
- Volume de backlog
- Délais moyens de traitement
- Respect des SLA
- Causes de blocage

## Outils
- Google BigQuery (SQL)

## Auteur
Data Analyst – Banque / Assurance (France)
